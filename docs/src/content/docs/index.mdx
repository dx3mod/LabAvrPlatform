Лабораторная платформа для организации полнофункциональной разработки под микроконтроллеры AVR 
на языках программирования C, C++ и ассемблер. 

###### Функционал 

- Создание проекта по шаблону: прошивки или библиотеки
- Сборка проекта с зависимостями
	* Автоматический resolving подключений  
	* Единое пространство имён 
	* Подключение в прошивку файлов (например, bitmap)
- Загрузка прошивки на МК (powered by avrdude)
- Интеграция с VScode

###### Особенности 

- Гибкая настройка
- Минимальная ручная конфигурация
